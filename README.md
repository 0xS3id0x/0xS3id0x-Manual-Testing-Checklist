# 0xS3id0x-Manual-Testing-Checklist

🛡️ Interactive web app for bug bounty hunters &amp; pentesters.
> 🔗 [Live Demo](https://0xs3id0x.github.io/0xS3id0x-Manual-Testing-Checklist/)

## ✨ Features

- ✅ **200+ vulnerability checks** across 20+ categories
- 🔍 **Real-time search & filtering** by severity or keyword
- 💾 **Offline-first**: All data saved in your browser — zero tracking
- ⌨️ **Keyboard shortcuts**: `/` search, `F` cycle filters, `R` reset
- 🎨 **Cyberpunk UI**: Dark mode, scanlines, optimized for long sessions

## 🚀 Quick Start

### Online (No Install)
1. Visit: [Live Demo](https://0xs3id0x.github.io/0xS3id0x-Manual-Testing-Checklist/)
2. Start hunting — progress auto-saves locally

```bash

# Open in browser or serve locally
